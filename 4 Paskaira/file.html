Terminalu komandos

  • cd- change directory - vaiksciojimas tarp aplankalu.
  • ls - kist files and directories - isvardinti failus ir aplankus
  • CTRL C - atsaukia vykdoma veiksma
  • TAB - uzbaigia fraze arba pateikia pasiulymus
  • clear - isvalyti(nujelia i virsu slamsta)

Git komandos

  • git init - initialized
  • git status - statusas, parodo dabartinia git main info su failais repozitorijoje
  • git add - stagina paruosia failus commitinimui
  • commit - sukuria kodo versija is stageintu failu
  • push - lokalius jau sukomitintus pakitmus perduoda git hub
